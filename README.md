# なにこれ

docxで書かれたPythonスクリプトを実行します．

# 必要なもの

* Python 3.x
* [tika-python](https://github.com/chrismattmann/tika-python)
* [Beautiful Soup](https://www.crummy.com/software/BeautifulSoup/)

# 使い方

Pythonスクリプトを実行

```
ugokudocx document.docx
```

ファイルに書き出し

```
ugokudocx document.docx -o output.py
```
